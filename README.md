# Multi-Asset Crypto Portfolio Management Dashboard

## 📌 Project Overview
An executive-level financial tracker designed to manage and evaluate multi-asset cryptocurrency portfolios. Built utilizing integrated workflows between Groq AI, Manus, and Excel, this model tracks historical trends, moving average momentum signals, and real-time net worth valuations across major digital tokens (BTC, ETH, BNB, SOL, XRP, ADA, DOGE).

## 📊 Features & Architecture
* **Executive Dashboard:** Clear summary matrix capturing net portfolio valuations and highlighting top-performing assets.
* **Technical Trend Analysis:** Structured historical pricing layers including 7-day (`SMA_7`) and 30-day (`SMA_30`) Simple Moving Averages.
* **Portfolio Valuation Engine:** Live holding calculations scaling asset allocation tokens directly against closing values.
* **Comprehensive Documentation:** Self-contained project architecture documentation tracking core logic parameters and version history within the engine.

## 🗂️ Sheet Structure
1. `Dashboard`: High-level summary blocks for corporate reporting.
2. `Charts_Data`: Clean calculation and staging layer for visualization engines.
3. `Raw Data`: Comprehensive transaction entries and asset trend lines.
4. `Project Documentation`: Architecture and asset coverage summaries.
